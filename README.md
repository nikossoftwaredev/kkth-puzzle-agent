## 🔐 Μέθοδοι Κρυπτογράφησης

| Keyword/Trigger              | Μέθοδος           | Περιγραφή                            |
| ---------------------------- | ----------------- | ------------------------------------ |
| Σησάχ, Βαβυλώνα              | Atbash            | Α↔Ω, Β↔Ψ, Γ↔Χ... Καθρέφτης αλφαβήτου |
| Καίσαρας, Ρώμη               | Caesar Cipher     | Μετατόπιση γραμμάτων κατά Ν θέσεις   |
| Καφεμαντεία, ελληνικός καφές | Ανάποδα           | Γύρνα το αντικείμενο ανάποδα         |
| Μορς, τελείες/παύλες         | Morse Code        | .- = A, -... = B, κλπ                |
| Τυφλός, κουκίδες             | Braille           | Σύστημα ανάγνωσης τυφλών             |
| Αριθμοί                      | Θέση στο αλφάβητο | 1=Α, 2=Β, 3=Γ...                     |
| Πληκτρολόγιο                 | Keyboard cipher   | Αντιστοίχιση ελληνικών-αγγλικών      |

## 📂 Κατηγορίες Γρίφων

### 1. 🔤 Ciphers (Κρυπτογραφήματα)

- Atbash
- Caesar
- Αντικατάσταση γραμμάτων
- Morse
- Braille
- Αριθμοί → Γράμματα

### 2. 👁️ Visual Puzzles

- Κρυμμένες εικόνες/λέξεις
- Οπτικές ψευδαισθήσεις
- Καθρέφτης/Ανάποδα
- Βρες τις διαφορές
- Στήλες/Γραμμές - κοινά στοιχεία

### 3. 🔤 Wordplay

- Αναγραμματισμοί
- Ομόηχες λέξεις
- Διπλές σημασίες
- Λογοπαίγνια

### 4. 🖐️ Physical Interaction

- Γύρνα ανάποδα
- Δίπλωσε το χαρτί
- UV φως
- Καθρέφτης
- Ζύγισε/Μέτρα

### 5. 🔍 Google Search Required

- Ιστορικές αναφορές
- Βιβλικές αναφορές
- Keywords → Απάντηση

### 6. 🧮 Pattern Recognition

- Βρες το κοινό στοιχείο
- Ακολουθίες
- Σχέσεις μεταξύ στοιχείων

### 7. 🧩 Συνδυασμός

- Πολλαπλά βήματα

## 💡 Tips & Patterns

| Αν δεις...                    | Σκέψου...                       |
| ----------------------------- | ------------------------------- |
| Ποτήρι καφέ, καφεμαντεία      | Γύρνα ανάποδα, διάβασε από κάτω |
| Σησάχ, Βαβυλώνα               | Atbash cipher                   |
| Καίσαρας, Ρώμη, λεγεώνα       | Caesar cipher                   |
| Ξ ή = μέσα σε κείμενο         | Εξίσωση, διαχωριστικό           |
| Αριθμούς                      | Θέσεις γραμμάτων στο αλφάβητο   |
| Στήλες με γράμματα            | Βρες τα κοινά σε κάθε στήλη     |
| Ανάποδο/καθρεφτισμένο κείμενο | Καθρέφτης ή φυσικά γύρνα        |
| Τελείες και παύλες            | Morse code                      |
| 6 κουκίδες σε pattern         | Braille                         |
| Περίεργη γραμματοσειρά        | Μπορεί να κρύβει κάτι           |
| QR code / Barcode             | Σκάναρε το                      |
